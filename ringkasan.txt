membuat ssh key (github)
* ssh-keygen -> maka akan muncul key public dan key private di folder .ssh yang telah ditentukan,
file tersebut biasanya bernama id_rsa dan id_rsa.pub
* setelah itu setting di server repository/key
* untuk menampilkan ssh keynya masuk kedalam .ssh dan ketik perintah cat id_rsa.pub (Linux)
* setelah copy key tersebut dan paste di setting key server
* maka otomatis akan terhubung

Membuat remot
* git remote add namaRemote ssh-url
ket : membuat remote ke server repo

* git remote
ket : menampilkan nama remot

* git remote get-url namaRemote
ket : mendapatkan nama url di server repo

* git remote rm namaRemote
ket : menghapus remot repository

* git push namaRemote localBranch
ket : push data ke server repo

* git push namaRemote localBranch:branchBaru
ket : push data ke server repo dengan namaBranch yang baru

* git push origin --all
ket : mengupload semua branch yang ada di localBranch

* git push --delete namaRemote namaBranch
ket : menghapus branch di repo server

* git clone namaUrlrepo di server
ket : mengclone repo yang ada di server termasuk dengan remote dan inizilisasi gitnya (tidak termasuk branch didalamnya)

* git clone namaUrlrepo namaFolder
ket : mengclone repo dari server namun menggunakan nama sesuai yang diinginkan

* git branch -r 
ket : melihat branch yang ada di remote repository

* git branch -a
ket : melihat semua branch yang di local dan remote

* git checkout -b namaBranchLocal namaRemote/namaBranchRemote
ket : mengambil data branch yang ada di remote dengan membuat branch di local

* git pull namaRemote namaBranch
ket : mengambil data di server repo dan mengubah di local
